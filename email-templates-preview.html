üîó How the Forgot Password Flow Works

Complete User Journey:

  1. User visits: localhost:5173/forgot-password (or your deployed URL)
  2. Submits email: Frontend calls resetPassword(email)
  3. Supabase sends email: Uses the "Password Reset" template below
  4. User clicks link: Redirects to /reset-password page
  5. User sets new password: Account is updated
